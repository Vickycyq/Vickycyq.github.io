  • 对新用户、不活跃用户、回流客户、活跃客户的定义
    • 若本月没有消费：
      • 是第一个月，将客户标记为‘未注册’
      • 不是第一个月：
        • 上个月‘未注册’，本月也‘未注册’
        • 上个月‘不活跃’，本月‘不活跃’
    • 若本月消费了：
      • 是第一个月，标记为‘新客户’
      • 不是第一个月：
        • 上个月不活跃，这个月‘回流’
        • 上个月‘未注册’，这个月‘新客户’ -上个月‘活跃’，这个月‘活跃’

根据以上定义，可以写下面的函数：

def active_status(df):
    status=[] #代码定义了一个空的列表 status，用于存储每个月的客户活跃状态
    for i in range(11): #通过一个循环从0到10（代表11个月，因为列的索引从0开始）遍历每个月的数据
    #若本月没有消费
    if df[i]==0:
        if len(status)>0: #如果不是第一个月
            if status[i-1]=='未注册‘: #如果上个月已经是未注册，那么本月也是未注册
                status.append('未注册')
            else: #如果上个月已注册，则本月为不活跃
                status.append('不活跃')
        else:
            status.append('未注册')
    else:
        if len(status)==0:
            status.append('新客户')
        else:
            if status[i-1]=='不活跃': #如果上月为不活跃，则本月为回流
                status.append('回流')
            elif status[i-1]=='未注册': #如果上月为未注册，那么本月为新注册
                status.append('新客户')
            else: #如果上月为活跃，本月也活跃
                status.append('活跃')
return pd.Series(status,index=columns_month)

3.2.7RFM模型构建#

  • R：最近一次消费（Recency）
  • F：消费频率（Frequency）
  • M：消费金额（Monetary）

按照下面代码构建RFM模型

user_rfm=df.groupby(['user_id','date']).agg(消费金额=('amount','sum')).reset_index()
user_rfm=user_rfm.groupby('user_id').agg(最后购买日期=('date','max'),M=('消费金额','sum'),F=('date','count')).reset_index()
user_rfm['最后购买日期'] = pd.to_datetime(user_rfm['最后购买日期'])
user_rfm['R'] = user_rfm['最后购买日期'].apply(lambda x:user_rfm['最后购买日期'].max() - x)
user_rfm['R'] = user_rfm['R'].dt.days
user_rfm=user_rfm[['user_id','R','F','M']]


#R的打分:
#[0-30]:5分
#(30-60]:4分
#(60-90]:3分
#(90-120]:2分
#120以上:1分#

#F的打分:
#1次:1分
#2次:2分
#3次:3分
#4次:4分
#5次及以上:5分

#M的打分:
#[0-200]:1分
#(200-500]:2分
#(500-1000]:3分
#(1000-2000]:4分
#2000以上:5分
rfm_score = user_rfm.copy()
